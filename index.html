App Tracker — Digital Wellbeing Monitor ▼

Repo
Android app that monitors usage patterns, detects addictive behavior, and manages screen time

A full-featured Android app that tracks foreground app usage in real time and alerts users when they exceed configurable time limits. A foreground service polls UsageStatsManager every 5 seconds to detect the active app, and sessions are persisted to a Room database. The addiction detector runs on a 7-day rolling window and scores apps (0–4) based on daily usage over 1 hour, frequent opens (20+/day), compulsive short sessions, and late-night usage between 11 PM and 6 AM.

Built with MVVM architecture using Hilt for dependency injection and Kotlin Coroutines with StateFlow for reactive data. Designed a fullscreen overlay warning system with escalating "Add Time" increments and a 60-second auto-dismiss safety net. Implemented a searchable settings screen with per-app sliders and a 3-step confirmation flow for excluding addictive apps. Created a weekly analytics dashboard with bar charts showing 7-day trends and a top-10 most-used ranking.

Kotlin Jetpack Compose Room DB Hilt MVVM Material Design 3

MOS 6502 CPU Emulator ▼

Repo
Software emulator for the classic 8-bit microprocessor used in the NES and Apple II

The emulator faithfully replicates the MOS 6502 CPU at the instruction level. It models all internal state — registers (A, X, Y), the program counter, stack pointer, and all processor status flags (N, V, Z, C, D, I) — and simulates a full 64 KB memory map.

Implemented all 151 official opcodes with a switch-based instruction decoder and isolated addressing mode handlers for Immediate, Zero Page, Absolute, Indexed (X/Y), Indexed Indirect, and Indirect Indexed. Built correct ADC/SBC logic including BCD decimal mode. Structured the codebase around a modular CPU core that cleanly separates instruction behavior, addressing modes, and state management for deterministic, testable execution.

C++ CPU Architecture Emulation Low-Level Arithmetic

Binary ALU Simulator ▼

Repo
Arithmetic-Logic Unit built from first principles using simulated logic gates

A console application that performs m,n-bit binary arithmetic on string-based inputs, demonstrating the fundamental logic used in CPU hardware. The calculator supports addition, subtraction, multiplication, division, modulo, and exponentiation on binary numbers of any length.

Built the core adder by simulating XOR, AND, and OR gates, then chained them into an n-bit ripple-carry adder. Implemented subtraction via two's complement, multiplication using the classic shift-and-add algorithm, and division using the restoring (shift-and-subtract) algorithm that iterates bit-by-bit while maintaining a partial remainder. Added exponentiation by converting the binary exponent to decimal and performing repeated multiplication through the existing multiply unit.

C++ Digital Logic Computer Architecture Gate Simulation

BMP Circle Generator ▼

Repo
Generates BMP images from scratch with no external libraries

A C++ program that produces valid 24-bit BMP image files entirely from raw binary data — no image libraries involved. The output is a 100×100 pixel image with a red circle centered on a white background.

Manually constructed the 14-byte BMP file header and 40-byte DIB info header using packed structs to guarantee correct binary layout. Iterated over a pixel grid and applied the circle equation (x−cx)² + (y−cy)² ≤ r² to classify each pixel as inside or outside. Handled row padding to 4-byte boundaries as required by the BMP spec and wrote the final output using reinterpret_cast and pointer arithmetic for precise memory-level control.

C++ Binary I/O File Formats Memory Layout
