MOS 6502 CPU Emulator ▼

Repo

A software-based emulator for the MOS 6502 8-bit microprocessor, accurately modeling the internal state, instruction execution pipeline, and memory access behavior of the historic hardware used in the NES, Apple II, and Commodore 64.

Key Contributions & Architecture
  • Instruction-Level Accuracy: Implemented all 151 official MOS 6502 opcodes, accurately simulating the full fetch–decode–execute cycle, including stack operations, register updates, and status flag behavior.
  • Memory & Addressing Modes: Modeled the full 64 KB memory map and supported all major 6502 addressing modes for deterministic program execution.
  • Modular Software Design: Architected the emulator in C++ with a clear separation between the CPU core, instruction decoder, and memory subsystem to maximize code maintainability and testability.

  • Domain Expertise Computer Architecture, Instruction Set Architecture (ISA), CPU State Management, Low-Level Systems Programming, Memory Mapping
